Reporting

Turn source tables into a reviewable Word report.

Build a DOCX draft from spreadsheets, CSVs and readable PDFs.

Give it a folder of source tables and a report type. Get back a Word draft, Markdown version, reading checks and review files.

Give it / get back

  • Source files: workbooks, CSV exports and readable PDFs.
  • Report choices: type, language and section assignments.
  • Output: DOCX draft, Markdown version, reading checks and review files.

From source files to a report you can review

The page keeps the practical sequence clear: inspect inputs, confirm where each table belongs, then draft a document with traceable sources.

Find tables

Creates inspection.json and suggested_recipe.json so the usable inputs are visible before drafting.

Assign sections

Keeps uncertain tables unassigned until the section choice is clear.

Draft report

Builds report.docx, report_draft.md, reading checks and review files from the confirmed section choices.

Reviewable outputs

Open report.docx first. Use report_analysis.json and report_audit.json to check how tables were assigned and what remains uncertain.

inspection.json suggested_recipe.json report_tables.json report_analysis.json report_draft.md report.docx report_audit.json used_recipe.json

Ready prompts

Start with the folder, report type and language. Ask for missing section choices only when they affect the draft.

  • Relazione ente locale Usa Build Report sui file in /percorso/ente. Lingua: it. Tipo report: local_government_review. Assegna FPV, FCDE, debito, cassa e investimenti, poi genera bozza Word e controlli di lettura.
  • Annual financial statement Use Build Report on this ZIP of annual accounts. Report type: annual_financial_statement. Keep uncertain tables unassigned, fill only supported comments, and produce report_analysis.json and report.docx.
