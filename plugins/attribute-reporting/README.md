# Attribute Reporting

[Source code](https://github.com/fabioannovazzi/app_files/tree/main/plugins/attribute-reporting) · [GNU AGPLv3 License](https://github.com/fabioannovazzi/app_files/blob/main/LICENSE)

Codex workflow with local report artifacts and an authenticated Mparanza evidence bridge for product attribute mapping, cohort analysis, Brand Fit, HTML reporting, and evidence-backed correctness review.

Brand Fit starts from a completed checked local Retailer Signals report, then uses an authenticated server package to compare those signals with the brand's current presence at that retailer and its owned catalogue. The source report, downloaded product images, and authored HTML report remain local. Codex performs semantic interpretation and independent review through the user's existing ChatGPT plan; plugin scripts make no separate model API call.
