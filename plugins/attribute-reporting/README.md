# Attribute Reporting

[Source code](https://github.com/fabioannovazzi/app_files/tree/main/plugins/attribute-reporting) · [GNU AGPLv3 License](https://github.com/fabioannovazzi/app_files/blob/main/LICENSE)

Local Codex workflow for product attribute mapping, cohort analysis, Brand Fit, HTML reporting, and evidence-backed correctness review.

Brand Fit starts from a completed checked local Retailer Signals report, then uses an authenticated server package to compare those signals with the brand's current presence at that retailer and its owned catalogue. The source report, downloaded product images, authored HTML report, and independent review remain local; plugin scripts call no model API.
