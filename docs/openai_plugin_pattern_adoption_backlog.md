# OpenAI Plugin Pattern Adoption Backlog

Use this backlog with `docs/openai_plugin_lessons_playbook.md` and
`docs/plugin_ui_migration_strategy.md`. It tracks what has already been
adapted from OpenAI plugin patterns and what still needs implementation in the
Mparanza local/deterministic setup.

## Status Legend

- `Adopted`: implemented in the relevant plugin family and covered by normal
  workflow docs or code.
- `Partial`: present, but not yet consistent enough to treat as a repo-wide
  standard.
- `Next`: high-priority implementation target.
- `Deferred`: useful later, but not needed for the next plugin quality step.
- `Rejected`: not compatible with the Mparanza model.
