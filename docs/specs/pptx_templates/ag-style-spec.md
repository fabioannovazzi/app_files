# A&G PPTX Style Spec

## Purpose

Canonical A&G consulting style definition for PPTX decks.

This spec is independent from other consulting presets. Do not borrow palette,
accent, or formatting rules from unrelated deck styles.

## Color Rules

- Always use the A&G deck theme.
- The theme has exactly three colors:
  - Blu: `#002060`
  - Azure blue: `#0070C0`
  - Light blue: `#00B0F0`
- Color order is mandatory:
  - If the deck needs only one color, use Blu `#002060`.
  - If the deck needs two colors, use Blu `#002060` and Azure blue `#0070C0`.
  - Use Light blue `#00B0F0` only when the deck genuinely needs three colors.
- Use shades of grey only when additional neutral hierarchy is needed, such as
  body text, table rules, muted labels, separators, or subdued backgrounds.
- Allowed greys:
  - Medium grey: `#666666`
  - Neutral grey: `#808080`
  - Light grey: `#BFBFBF`
  - Rule grey: `#D9D9D9`
  - Background grey: `#F2F2F2`
- Black `#000000` is not a grey. Use it only where explicitly required, such as
  slide titles.
- Do not introduce any other colors.
- Do not use decorative gradients, arbitrary accents, traffic-light colors, or
  non-theme chart palettes.

## Typography Rules

- Use Verdana as the font throughout A&G PPTX decks.
- Use 24pt for slide titles.
- Slide titles must be plain black (`#000000`) text on a white background.
- Slide titles are not bold by default.
- Use bold inside slide titles only to highlight selected words or short
  phrases when it improves the message.
- Do not place slide titles in colored title banners or filled blocks.
- Use 14pt for ordinary slide text.
- Ordinary slide text may range from 12pt to 16pt when needed for hierarchy or
  fit.
- Disclaimers may use smaller text when necessary.
- Do not mix in other font families for titles, body text, tables, captions, or
  chart labels unless a supplied A&G template explicitly requires it.
- Never make one label smaller than similar labels just to make it fit inside an
  element. Similar labels must keep the same font size.
- PowerPoint text boxes must use "Do not autofit" / "Non adattare".
- Do not use "Shrink text on overflow" / "Adatta testo nell'area di
  riversamento", because it changes the font size and breaks typography
  consistency.
- If text does not fit, rewrite the text, enlarge the text box, or change the
  layout. Do not solve fit by automatic font shrinking.

## Text Emphasis Rules

- Text should be normal weight by default.
- Use bold for the first level of emphasis.
- Use underline for a stronger emphasis level after bold.
- Italic is allowed when it carries a clear editorial or semantic purpose.
- Text blocks should never, or only very rarely, be entirely bold. If everything
  is bold, nothing is highlighted.
- Apply emphasis selectively to words or short phrases, not as a default style
  for whole paragraphs, table cells, or labels.

## Text Box Settings

Use these PowerPoint text box settings for A&G PPTX shapes:

- Vertical alignment: Bottom / `In basso`.
- Text direction: Horizontal / `Orizzontale`.
- Autofit: Do not autofit / `Non adattare`.
- Left margin: `0 cm`.
- Right margin: `0 cm`.
- Top margin: `0.13 cm`.
- Bottom margin: `0.13 cm`.
- Wrap text in shape / `Testo con a capo automatico nella forma`: enabled.

## Practical Application

- Default emphasis, section markers, primary rules, and primary chart series use
  Blu.
- Secondary emphasis or a second chart series may use Azure blue.
- Light blue is reserved for a third meaningful category, series, or hierarchy
  level. It should not appear just to make a slide feel more colorful.
- When a chart or exhibit has more than three categories, use the three theme
  blues first, then grey shades for the remaining categories if they must be
  visible.
- Neutral content should remain grey or near-black, not theme-colored by
  default.

## Visual Consistency Rules

- Identical visual elements repeated across successive slides must keep the same
  size.
- This applies to arrows, lines, icons or symbols when permitted, callout
  markers, dividers, labels, and other repeated shapes.
- Do not resize a repeated element from one slide to the next unless the change
  communicates a deliberate difference in meaning.
- Colored title blocks are allowed for card headers and comparable exhibit
  headers, not for the main slide title.
- In a group of cards on the same slide, card titles must use the same visual
  treatment.
- If the first card title uses a colored title block with white text, every
  comparable card title in that group must also use a colored title block with
  white text.
- For one-color A&G decks, card title blocks use Blu `#002060` with white text.
- Do not mix one card with a filled title block and other cards with plain text
  titles unless the difference communicates a deliberate hierarchy.

## Do Not

- Do not use Bain red, governance green, warm neutrals, or any palette from
  another template.
- Do not treat the three blues as interchangeable decoration.
- Do not use Light blue when Blu alone, or Blu plus Azure blue, is enough.
- Do not add color purely for visual variety.
